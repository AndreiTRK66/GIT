# 🔐 Flutter Firebase Auth App

A basic Flutter app with Firebase Authentication integration, featuring:

- ✅ Register users with email & password
- 🔐 Email verification flow
- 🔓 Login only after email verification
- 🔄 Navigation between login/register/verification views
